Fundamentos do HTML & CSS

HTML (HyperText Markup Language) é a linguagem de marcação que estrutura o conteúdo das páginas web. É como o esqueleto de uma casa - define onde cada elemento deve estar posicionado.

CSS (Cascading Style Sheets) é responsável pela apresentação visual. É como a decoração da casa - define cores, tamanhos, posicionamento e toda a aparência visual.

Exemplo: Estrutura HTML Básica

Aqui está um exemplo de documento HTML básico que demonstra a estrutura fundamental: 

<!DOCTYPE html>
<html lang="pt">
  <head>
    <meta charset="utf-8">
    <title>Olá, HTML!</title>
  </head>
  <body>
    <h1>Bem-vindo</h1>
    <p>Este é o teu primeiro parágrafo.</p>
  </body>
</html>
Explicação do Código:

<!DOCTYPE html> - Declara que este é um documento HTML5
<html lang="pt"> - Elemento raiz com idioma português
<head> - Contém metadados (informações sobre a página)
<meta charset="utf-8"> - Define a codificação de caracteres
<title> - Título que aparece na aba do navegador
<body> - Conteúdo visível da página

Conceitos Fundamentais

Semântica HTML

Use elementos HTML que descrevem o significado do conteúdo, não apenas a aparência. Por exemplo: <header>, <nav>, <main>, <article>, <section>, <footer>.

Acessibilidade (A11y)

Sempre inclua atributos alt em imagens, use headings hierárquicos (h1, h2, h3...) e garanta que o site seja navegável por teclado e leitores de ecrã.

CSS Moderno

Flexbox e CSS Grid são as ferramentas modernas para layout. Flexbox é ideal para layouts unidimensionais (linha ou coluna), Grid para layouts bidimensionais complexos.

Critérios de Avaliação

Aqui estão os critérios organizados por níveis de importância para avaliar o teu domínio dos fundamentos de HTML e CSS.

💠 Essencial

  • Estrutura HTML básica (DOCTYPE, html, head, body)
  • Tags semânticas principais (header, nav, main, footer)
  • Atributos fundamentais (id, class, src, href, alt)
  • Boas práticas de acessibilidade (A11y)
  • Hierarquia de headings (h1, h2, h3...)
  • Elementos de texto (p, strong, em, span)
  • Listas ordenadas e não ordenadas
  • CSS básico (seletores, propriedades, valores)

⭐ Importante

  • Formulários HTML (input, label, button)
  • Layout com Flexbox
  • CSS Grid básico
  • Responsividade com media queries
  • Pseudo-classes (:hover, :focus, :active)
  • Box model (margin, padding, border)
  • Posicionamento CSS (relative, absolute)
  • Unidades CSS (px, em, rem, %, vh, vw)

🚀 Avançado

  • Semântica avançada (article, section, aside)
  • Performance e otimização
  • Organização e metodologias CSS (BEM)
  • Integração básica com JavaScript
  • CSS Grid avançado
  • Animações e transições CSS
  • Custom properties (CSS variables)
  • Progressive enhancement
