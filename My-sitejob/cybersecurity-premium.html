Fundamentos de Cibersegurança

A Cibersegurança é a área que protege sistemas, redes e dados contra ataques digitais.

O objetivo é garantir a confidencialidade, integridade e disponibilidade da informação, prevenindo acessos não autorizados, falhas e roubos de dados.

Exemplo: Proteção de dados

Aqui está um exemplo: Guarda como password_hash.py e abre no VS Code. 


import hashlib

def criar_hash(password):
    hash_password = hashlib.sha256(password.encode()).hexdigest()
    return hash_password

password = input("Introduz a tua password: ")
print(criar_hash(password))
Explicação do Código:

<import hashlib> - importa a biblioteca para criar hashes criptográficos
<def criar_hash(password):> - Define uma função para criar um hash SHA256 de uma password
<password.encode()> - Converte a password para bytes (necessário para o hash)
<hashlib.sha256(...)> - aplica o algoritmo de hash SHA-256
<.hexdigest()> - Retorna o hash como uma string hexadecimal
<input()> - pede a password ao utilizador <print()> - imprime o hash da password
