🔍 O que é Web Development?

Cloud Computing permite aceder a servidores e serviços pela internet. Tem como pilares:

  • Infraestrutura como Serviço (IaaS)
  • Plataforma como Serviço (PaaS)
  • Contêineres e Orquestração (Kubernetes)
  • Armazenamento e Redes Cloud
  • DevOps e CI/CD

📚 Recursos para Estudar

  • OpenStack
  • Apache CloudStack
  • OpenNebula
  • CNCF - Cloud Native Foundation
  • Google Cloud Documentation
